# 🎥 WEZO - Professional Wedding Photography

موقع احترافي للتصوير الفوتوغرافي للأفراح مع تكامل واتساب للحجوزات

## ✨ المميزات الرئيسية

### 📱 نظام الحجوزات عبر واتساب
- نموذج حجز مرن وسهل الاستخدام
- إرسال تلقائي للرسائل على واتساب
- حفظ البيانات محلياً في المتصفح
- تصدير الحجوزات كـ CSV

### 🎨 التصميم الحديث
- واجهة استخدام احترافية وانجذابية
- تصميم ديناميكي يستجيب للجوال (Responsive)
- ألوان حديثة وتدرجات جميلة
- رسوم متحركة سلسة وعصرية

### 📸 معرض الصور
- ألبومات صور منظمة
- معاينة صور بتأثيرات جذابة
- عرض صور كامل الشاشة (Lightbox)

### 🌍 تحسين محرك البحث (SEO)
- عناوين وأوصاف ميتا محسّنة
- Open Graph للمشاركة على وسائل التواصل
- بنية HTML منطقية

## 🔧 الملفات الرئيسية

### `whatsapp-contact.js`
يحتوي على جميع وظائف التعامل مع واتساب:
- `generateWhatsAppLink()` - إنشاء رابط واتساب مخصص
- `handleContactForm()` - معالجة البيانات من الاستمارة
- `saveFormData()` - حفظ البيانات محلياً
- `getAllBookings()` - الحصول على جميع الحجوزات
- `exportBookingsAsCSV()` - تصدير البيانات

### `contact-form.html`
صفحة استمارة الاتصال المتقدمة بميزات:
- تصميم جميل وعصري
- حقول منظمة وواضحة
- معالجة أخطاء ورسائل تأكيد

## 📞 نظام الاتصال

### كيفية عمل الحجز:
1. المستخدم يملأ النموذج
2. البيانات تُحفظ محلياً
3. يتم إنشاء رسالة واتساب مفصلة
4. يتم فتح واتساب تلقائياً بالرسالة

### رقم الواتساب
```javascript
+20 1033964828  // الرقم الافتراضي
```

يمكن تغييره في `whatsapp-contact.js`

## 🎯 أقسام الموقع

- **الرئيسية (Hero)** - مقدمة جذابة مع دعوة للعمل
- **من نحن** - معلومات عن الفريق والإحصائيات
- **الألبومات** - معرض الصور والأعمال السابقة
- **الخدمات** - تفاصيل الخدمات المقدمة
- **آراء العملاء** - شهادات العملاء الراضين
- **التواصل** - النموذج والمعلومات الاتصال

## 💾 إدارة البيانات

### حفظ البيانات محلياً

```javascript
getAllBookings()

// التصدير إلى CSV
exportBookingsAsCSV()
```
